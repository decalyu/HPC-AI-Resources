## 🖥️ 1️⃣ What is the Command Terminal?
A **command terminal** (or command line) is a tool used to interact with a computer by typing commands instead of clicking on icons. It allows you to control your system more efficiently.

## 🔥 Why Learn It?
- Used by AI researchers, software developers, and engineers
- Essential for running programs on powerful computers (HPC)
- Helps automate repetitive tasks
